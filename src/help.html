HELP MENU



General Utility

Help Button : To open this help menu

Reset Button : To reset the object to its original state and position

Save Button : To save the current object as JSON file

Load Button : To load the object from JSON file



Choose Provided Object

Click the button below the canvas to choose the provided object that you want to load

There are 3 objects provided : cube, pyramid, interlocking cube



Change Projection Type

Choose the type of projection that you want to use: orthographic or oblique or perspective



Transform the Object

Translation : Use the slider to translate the object along x, y, z axis

Rotation : Use the slider to rotate the object about the x, y, z axis

Scaling : Use the slider to scale the object in x, y, z axis



Set the Camera

Camera Look At : Use the slider to rotate the camera about the x, y, z axis

Camera Radius : Use the slider to zoom in or out the camera



Change Object's Color

Use the color picker to change the color of the object



Shading Option

Press the switch to enable or disable the shading of the object

The shading is enabled by default
